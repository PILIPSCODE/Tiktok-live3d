# Console

`Console` is a feature for monitoring all livestream activities in real time.  
It displays logs that help developers and hosts track events during the session.

---

## User Joined Log
Logs when a new user joins the livestream.  

## Share Log
Logs when the livestream is shared. 


## Follow Log
Logs user


## AI Response Log
If AI integration is enabled, the console also displays **AI responses** in real time.
