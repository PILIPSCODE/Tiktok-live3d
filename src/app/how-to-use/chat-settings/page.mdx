# Chat Settings Documentation

## Overview

Chat Settings provides comprehensive customization options for chat interface appearance and behavior. This documentation covers text alignment, border styling, and default speak functionality.

## Features

- **Text Alignment**: Customizable message, username and comment text alignment
- **Border Styling**: Configurable chat bubble borders
- **Default Speak**: Automatic text-to-speech functionality (unstable)
